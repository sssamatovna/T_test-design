# Блок Т: Теория тестирования 🎯

Данный репозиторий содержит цикл практических работ по модулю «Теория тестирования». Проект разделен на две ключевые задачи, охватывающие проектирование тестов (Test Design) и работу с дефектами (Bug Reporting).

## 📋 Содержание проекта

1. Задача Т1: Проектирование тестов для API получения статуса продукта.
2. Задача Т2: Тестирование требований и написание баг-репортов для Яндекс.Диска (UI & API).
  
**(Задачи их их описания будут появляться по мере решения.)**
